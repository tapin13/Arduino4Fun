# Arduino - IR Read Sensor

## Detect barrier/object between two points.

Used items:
* BreadBoard
* Arduino Uno
* IR Transmitter
* IR Receiver
* Resistor 100K ohm (for IR Receiver)
* Resistor 220 ohm (for IR Transmitter)
